# Portfolio Description

## Rental Property Management & Analytics System

> End-to-end data analytics project for a real-world rental property, built with Python, SQL, and Streamlit.

---

### What I Built

A complete analytics system for a 3-story townhouse rental property in Taiwan — 9 residential units, 1 commercial storefront, and 5 scooter parking slots. The system covers the full data lifecycle: **schema design → data loading → validation → analytics → visualization**.

### Why It Matters

Property owners make pricing, tenant, and maintenance decisions based on intuition. This project replaces guesswork with data — tracking occupancy patterns, identifying payment risk, monitoring expense trends, and calculating profitability per unit.

### Technical Highlights

| Area | What I Did |
|------|-----------|
| **Data Modeling** | Designed 8-table normalized schema with foreign keys and constraints |
| **SQL** | 20+ queries with CTEs, conditional aggregations, multi-table joins |
| **ETL** | Python pipeline: CSV → SQLite with dependency ordering |
| **Data Quality** | 25 automated checks (integrity, nulls, ranges, duplicates, business rules) |
| **Analytics** | 10+ KPIs: occupancy, NOI, payment rates, rent/ping, cash flow |
| **Visualization** | Streamlit dashboard with Plotly charts and interactive filters |
| **Documentation** | Full README, project summary, resume bullets, interview points |
